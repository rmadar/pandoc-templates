6.2 Pandas as default and Markdown

import pandas as pd
df=pd.DataFrame(np.random.randn(10,3))

Default printing is HTML, so it looks good on the web but it is not well rendered in pdf via ipynb->MB->pdf (using nbconvert and pandoc). A special function df2md(df) is included in the jupy_pandoc_utils package to write out a markdown format. This allows to set caption and even to refer to the table in the main document like this (using vanilla pandoc) or cited like table Tbl. 2 (using pandoc-crossref filter)
